Spectral mesh analysis with the graph Laplacian.

FourierMesh computes mesh Fourier modes from the graph Laplacian, reconstructs geometry from low-frequency eigenvectors, and exports cleaned STL meshes for downstream geometry and 3D-printing workflows.

How It Works Install
Graph Laplacian

Mesh Fourier Transform

For STL triangle meshes, FourierMesh treats vertices as graph nodes and triangle edges as connectivity. Laplacian eigenvectors become mesh Fourier modes, ordered from smooth low frequencies to detailed high frequencies. Keeping the first k modes reconstructs a smoothed surface.

L = D - W
L u_i = lambda_i u_i | coeffs = U.T @ V | V_hat = U[:,1:k] @ coeffs[1:k]

David Statue Reconstruction

The David example uses the mesh Laplacian path: load an STL, deduplicate vertices, construct connectivity from triangle faces, compute low-frequency graph eigenvectors, reconstruct vertex positions with the first k modes, clean the mesh, and export a new STL for downstream tools.

Use Cases

  1. Mesh smoothing and low-pass filtering over connectivity.
  2. Spectral compression by keeping the first k Laplacian modes.
  3. Geometry-processing demos with quantitative reconstruction error.
  4. STL export for slicers after cleanup and optional trimesh repair.

Core APIs

load_mesh_stl, save_mesh_stl, mesh_laplacian_eigenmodes, mesh_fourier_transform, inverse_mesh_fourier, and reconstruct_mesh.

Runtime Profile

Dense Laplacian eigendecomposition costs O(N^3) and is used only for small meshes. Larger inputs use sparse eigsh low-mode solves. The David example supports --max-faces for fast interactive runs.

Project Status

FourierMesh v0.2 is a beta-stage scientific Python package for mesh spectral analysis. Runtime: Python 3.10+, NumPy, SciPy, NumPy-STL. Optional: Matplotlib demos and Trimesh slicer repair. Legacy point-cloud Dirac code is on the dirac branch.

Quick Start

from FourierMesh import load_mesh_stl, reconstruct_mesh

vertices, faces = load_mesh_stl("model.stl")
v_smooth, lambdas = reconstruct_mesh(vertices, faces, k=24)
